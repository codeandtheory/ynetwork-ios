URLProtocolStub

final public class URLProtocolStub : URLProtocol

URLProtocolis an abstract class. To intercept requests we need to create a subclass of URLProtocol. Required methods when subclassing: class func canInit(with request: URLRequest) -> Bool class func canonicalRequest(for request: URLRequest) -> URLRequest func startLoading() func stopLoading()
