NetworkRequest

public protocol NetworkRequest

All the information needed to make a network request. The only required property to implement is path. Everything else has default values.

  • basePath Default implementation

    Optional base path of the request. May be nil if path is an absolute path or if NetworkManagerConfiguration.basePath has been set. If non-nil, supercedes NetworkManagerConfiguration.basePath. Default nil

    Default Implementation

    Uses no base path

  • usesSession Default implementation

    Whether this request should use session information. Default true

    When true the network manager’s associated session manager (if any) may adjust the request (typically by adding a session token to the request’s headers) prior to submiting, and in the event of a 401 response will attempt to refresh the session and retry the request.

    When false, the session manager will not adjust the request nor retry on a 401 response.

    Typically you would want to return false for your refresh token request and for other unauthenticated requests (such as log in, sign up, or register requests).
