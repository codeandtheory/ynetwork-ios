Type Aliases

The following type aliases are available globally.

  • APICompletion

    Represents an asynchronous completion callback for a network request. The callback will supply either .success + the expected response object or else .failure + an Error.

    Declaration

    Swift

    public typealias APICompletion<Response> = (Result<Response, Error>) -> Void where Response : Decodable
  • HttpHeader

    Represents a single HTTP header

    Declaration

    Swift

    public typealias HttpHeader = (field: String, value: String)
  • HttpHeaders

    Represents a dictionary of HTTP headers

    Declaration

    Swift

    public typealias HttpHeaders = [String : String]
  • NetworkEngineCompletion

    Represents an asynchronous completion callback for a raw network request. The callback will supply either the data and the URL response or else an error.

    Declaration

    Swift

    public typealias NetworkEngineCompletion = (Data?, URLResponse?, Error?) -> Void
  • Parameters

    Represents a parameters dictionary. Typically this will be used to form the urlQuery component of the complete url path for a network request (typically a GET request).

    Declaration

    Swift

    public typealias Parameters = [String : Any]
  • Percentage

    Percentage expressed as a double value

    Declaration

    Swift

    public typealias Percentage = Double
  • ProgressHandler

    Asynchronous completion handler that reports file progress as a percentage (ranging from 0.0 to 1.0).

    Guaranteed to be called back on the main thread

    Declaration

    Swift

    public typealias ProgressHandler = (Percentage) -> Void
  • FileDownloadHandler

    Asynchronous completion handler that reports the status of request.

    Guaranteed to be called back on a background thread (because the system erases the temporary file)

    Declaration

    Swift

    public typealias FileDownloadHandler = (Result<URL, Error>) -> Void
